Case Study: My First Full-Stack Project - myFlix

Overview

myFlix is a full-stack movie website developed using the MERN stack. The platform enables users to:

  • Sign up and manage their accounts.
  • Access detailed movie information, including directors, genres, and other features.
  • Update personal data and maintain a list of favorite movies.

This project showcases my ability to design and implement both front-end and back-end systems from scratch.

Purpose

I developed myFlix as part of the CareerFoundry Bootcamp.

Objective: To gain hands-on experience building a complete full-stack application while preparing a polished portfolio piece.

Tools & Technologies

  • Back-End: Node.js, Express.js, MongoDB, Mongoose, Passport.js, JSON Web Token (JWT).
  • Front-End: React, JavaScript, CSS.
  • Testing: Postman.

Development Highlights

Server-Side

- Built a RESTful API using Node.js and Express.js to interact with a MongoDB database. - Implemented CRUD operations for movies and users. - Developed robust authentication and authorization using Passport.js and JWT. - API testing and validation using Postman.

Client-Side

- Designed a responsive front-end using React and CSS. - Integrated the front-end with the back-end API for seamless data exchange. - Overcame initial challenges with API integration and React-Redux state management, with guidance from my tutor.

Challenges & Lessons Learned

This project was my favorite because it introduced me to the complexities of back-end development and database management, which I thoroughly enjoyed. The most significant challenge was connecting the API to the front-end, but with persistence and support, I successfully resolved it.

Working on myFlix gave me invaluable insights into:

  • Structuring and managing full-stack projects.
  • React and state management using Redux.
  • Debugging and troubleshooting API connectivity issues.

Duration

Developing the client-side took twice as long as a standard CareerFoundry achievement due to the steep learning curve of React and Redux.
